# Exercise 3 - Stored Procedures

## Objective
Create and execute a stored procedure in MySQL.

## Software Used
- MySQL Server
- MySQL Workbench

## Steps
1. Create Database
2. Create Employee Table
3. Insert Sample Records
4. Create Stored Procedure
5. Execute Procedure

## Procedure

GetEmployeesBySalary(min_salary)

Returns all employees whose salary is greater than or equal to the given salary.

## Output

The procedure displays employees satisfying the salary condition.
